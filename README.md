# bootstrap-tree
Treely - Nova Creator Bootstrap Tree [URL] (http://novacreatorsoftware.github.io/bootstrap-data-tree/) 

* package.json =>
* npm install
* [this gets you stuff like grunt] 

* bower.json =>
* bower install
* [dependency manager]

* Gruntfile.json =>
* grunt build
* grunt release
* grunt buildcontrol:pages => update gh-pages (website)
* [release builder]
* ==> Dist folder has the release
